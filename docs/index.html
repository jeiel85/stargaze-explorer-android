주요 프리미엄 기능

최첨단 모바일 그래픽스 및 천문 지능이 결합한 완전히 새로운 차원의 별자리 관측 라이프

🧭

실시간 천체 AR 탐험 지도

모바일 자이로/나침반 센서와 CameraX 라이브 뷰파인더를 연동하여, 카메라 렌즈가 비추는 밤하늘에 은하 좌표계와 화려한 신화 속 별자리 기하선들을 정교하게 결합 오버레이합니다.

✨

Gemini 3.5 Flash 천문 코치

유효한 API 키가 연동되면 Gemini 3.5 Flash가 실시간 기온/구름/습도를 종합하여 감성적인 최적 관측 조언과 방한 대책을 생성합니다. 키가 없거나 오프라인일 때도 탄력적으로 로컬 가이드로 즉시 폴백합니다.

📓

Room 기반 관측 일기장 & 도감

내가 목격한 별자리 관측 로그와 소중한 밤하늘 현장 사진을 로컬 Room DB에 영구 보존합니다. 일기를 남길 때마다 도감 상의 별자리가 금빛 카드로 해제되며 성취감을 선물합니다.

👥

실시간 관측가 공유 커뮤니티

강릉 안반데기, 가평 화악산, 양평 벗고개 등 대한민국 유명 관측 명소의 실시간 날씨 인덱스와 전국의 모험가들이 업로드한 은하 사진들을 한눈에 공유하고 좋아요와 댓글로 소통합니다.

🔔

대유성우 및 월식 특보 알림

페르세우스 유성우 극대기나 블러드문 개기월식 등 놓쳐선 안 될 메이저 우주 일정들을 사전 터치 한 번으로 알람 예약하고, 피크 타임 기상 상황에 맞춰 지능형 푸시 특보를 수령합니다.

🔒

보안 및 로컬 오프라인 최우선

사용자의 민감한 개인정보나 관측 일기, 사진 데이터는 로컬 단말기에만 암호화 보관되며, 원격 클라우드 수동 동기화를 통해서만 안전하게 백업 통제됩니다.

로컬 로드맵 & 시작하기

단 3단계 만으로 나만의 천체 망원경 앱을 빌드하고 실행하세요

1

저장소 복제 및 임포트

Android Studio를 구동한 뒤 Git 저장소를 복제하거나 임포트합니다.

git clone https://github.com/jeiel85/stargaze-explorer-android.git
2

인공지능 Secrets 환경 변수 주입

프로젝트 루트 경로에 .env 파일을 생성하고 본인의 Google AI Studio API 키를 주입합니다. (키가 없으면 오프라인 탄력 모드로 작동합니다.)

GEMINI_API_KEY=AIzaSy...YourActualGeminiKey...
3

원스톱 릴리즈 빌드 및 데스크톱 추출

Gradle을 통해 App Bundle을 컴파일하고, 전용 우회 릴리즈 스크립트를 통해 Windows 바탕화면에 이식합니다.

.\gradlew.bat bundleRelease
powershell -ExecutionPolicy Bypass -File scripts\export-play-store-release.ps1
